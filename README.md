# JAXBase-WinForms
Early dev of JAXBase V1 for Windows only
  
## Important Notice
This is an early, work-in-progress prototype / migration staging area for JAXBase.<br>
This version is in-progress conversion from WinForms to Avalonia and is not the official release version.<br>
The new version well use the free Avalonia framework to make it Windows/Linux/Mac compatible.<br>
This code is experimental, incomplete, and in many cases IT WON'T WORK - Intended for development discussion and conversion planning only.<br>

If you really want to download it, feel free as it is open-source.<br><br>

See the main [repo](https://github.com/jlwalker-dev/JAXBaseV1) and [wiki](https://github.com/jlwalker-dev/JAXBaseV1/wiki) for the current source, project overview, language specs, and roadmap.
